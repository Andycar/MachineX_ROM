ପୋର୍ଟେବଲ୍ Wi-Fi ହଟ୍‌ସ୍ପଟ୍

ଆପଣ ଆପଣଙ୍କର ଡିଭାଇସକୁ ପୋର୍ଟେବୁଲ୍ Wi-Fi ହଟ୍‌ସ୍ପଟ୍‌କୁ ପରିବର୍ତ୍ତନ କରି, ଏହାର ଇଣ୍ଟର୍ନେଟ ଆପଣଙ୍କର କମ୍ପ୍ୟୁଟର୍ କିମ୍ଵା ଅନ୍ୟ ଡିଭାଇସ୍‌ଗୁଡିକ ସହିତ ଅଂଶୀଦାର କରିବାକୁ
  • ଡିଭାଇସ୍ ଆପ୍ଲିକେସନ୍ Wi-Fi ମାଧ୍ୟମରେ ଇଣ୍ଟର୍ନେଟ୍ ବ୍ୟବହାର କରିବାକୁ ଅସମର୍ଥ ହେବ ଯେତେବେଳେ ଡିଭାଇସ୍ ପୋର୍ଟେବୁଲ୍ Wi-Fi ହଟ୍‌ସ୍ପଟ୍ ଭାବେ ବ୍ୟବହାରରେ ଥିବ

  • ଆପଣ Wi-Fi ହଟ୍‌ସ୍ପଟ୍ ସେଟିଂଗୁଡିକରେ ପୋର୍ଟେବଲ୍ Wi-Fi ହଟ୍‌ସ୍ପଟ୍ ବିନ୍ୟାସ କରିପାରିବେ

  • ଅଧିକ ସୂଚନା ପାଇଁ, http://www.android.com/tether#wifi ପରିଦର୍ଶନ କରନ୍ତୁ

USB ଟିଥରିଂ

ଆପଣ ଆପଣଙ୍କ ଡିଭାଇସର ଇଣ୍ଟର୍ନେଟକୁ କମ୍ପ୍ୟୁଟରରେ ସଂଯୋଗ କରି ଅଂଶୀଦାର କରିପାରେ, ଯଦି ଆପଣ ତାହାକୁ ଏକ USB କେବୁଲ୍ ମାଧ୍ୟମରେ ଏକତ୍ର ଟିଥର୍ କରନ୍ତି
  • ୱିଣ୍ଡୋଜ୍ 7, ୱିଣ୍ଡୋଜ୍ ଭିଷ୍ଟା, ଏବଂ ଲାଇନକ୍ସ ସହିତ ଟିଥରିଂ କାର୍ଯ୍ୟ କରେ

  • ଆପଣଙ୍କ ଡିଭାଇସ୍ ଟିଥର୍ ହୋଇଥିବା ସମୟରେ ଆପଣ ଆପଣଙ୍କ ଡିଭାଇସର SD କାର୍ଡକୁ ଆପଣଙ୍କ PC ସହିତ ସଂଯୁକ୍ତ କରିପାରିବେ ନାହିଁ

  • ଅନ୍ୟ ଅପରେଟିଂ ସିଷ୍ଟମ ସହିତ USB ଟିଥରିଂ କିପରି ବ୍ୟବହାର କରିବେ ତାହା ଉପରେ ଅଧିକ ସୂଚନା ପାଇଁ, http://www.android.com/tether#usb କୁ ଯାଆନ୍ତୁ

ବ୍ଲୁଟୁଥ୍ ଟିଥରିଂ

ଆପଣ ବ୍ଲୁଟୁଥ୍ ମାଧ୍ୟମରେ ଆପଣଙ୍କର କମ୍ପ୍ୟୁଟରକୁ ଆପଣଙ୍କର ଆଣ୍ଡ୍ରୋଏଡ୍ ଡିଭାଇସ୍ ଟିଥର୍ କରି, କମ୍ପ୍ୟୁଟର ସହିତ ଏହାର ଇଣ୍ଟର୍ନେଟ୍ ସଂଯୋଜନ ଅଂଶୀଦାର କରିପାରିବେ
  • ବ୍ଲୁଟୁଥ୍ ଟିଥରିଂ ଯେକୌଣସି କମ୍ପ୍ୟୁଟର୍ କିମ୍ଵା ଡିଭାଇସ୍ ସହିତ କାର୍ଯ୍ୟ କରିବ ଯାହା ବ୍ଲୁଟୁଥ୍ ମାଧ୍ୟମରେ ଇଣ୍ଟରନେଟ ପ୍ରବେଶ କରିପାରୁଥିବ

  • ଅଧିକ ସୂଚନା ପାଇଁ, http://www.android.com/tether#bluetooth ପରିଦର୍ଶନ କରନ୍ତୁ
