पोर्टेबल Wi-Fi हॉटस्पॉट

आपण आपले उपकरण, त्याचे इंटरनेट कनेक्शन काँम्पुटर्स किंवा अन्य उपकरण बरोबर शेअर करण्यासाठी वाय-फाय हॉटस्पॉटमध्ये बदलू शकता
  • उपकरण Wi-Fi हॉटस्पॉट म्हणून काम करताना Wi-Fi कनेक्शन द्वारे इंटरनेट ऍक्सेस करण्यासाठी उपकरण अनुप्रयोग वापरण्यास अक्षम.

  • आपण Wi-Fi तेथरिंग सेटिंग्स मध्‍ये पोर्टेबल Wi-Fi तेथरिंग कॉन्फिगर करू शकता.

  • अधिक माहितीसाठी http://www.android.com/tether#wifi-ला भेट द्या

USB तेथरिंग

USB केबलबरोबर आपण त्यांना एकत्र टेथर केल्यास आपला कॉम्प्यूटर उपकरणाचे इंटरनेट कनेक्शन सहभागी करू शकतो.
  • टेथरिंग विंडोस7,विंडोस विस्टा, आणि लिनुक्ष बरोबर काम करते

  • टेथर केलेले असल्यास आपण SD कार्ड आपल्या कॉम्प्यूटरला कनेक्ट करू शकत नाही.

  • अधिक माहितीसाठी, USB टेथरिंग इतर ऑपरेटिंग सिस्टीमबरोबर कसे वापरावे यासह, अधिक माहितीसाठी, http://www.android.com/tether#usb-वर जा

ब्लूटूथ तेथरिंग

आपल्या कॉम्प्यूटरशी आपल्या उपकरणचे इंटरनेट कनेक्शन शेयर करण्यासाठी, आपण ब्लुटूथ द्वारे आपल्या कॉम्प्यूटरमध्ये आपले एनड्रॉइड उपकरण तेथर करू शकता.
  • ब्लुटूथ द्वारे इंटरनेट ऍक्सेस करू शकेल अशा कोणत्याही कॉम्प्यूटर किंवा उपकरण बरोबर ब्लुटूथ तेथरिंग काम करू शकते.

  • अधिक माहितीसाठी, पाहा http://www.android.com/tether#bluetooth
