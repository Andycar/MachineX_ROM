પોર્ટબલ Wi-Fi હૉટસ્પૉટ

તમે તમારા ડિવાઇસના ઇન્ટરનેટ જોડાણને કમ્પ્યુટરો અથવા અન્ય ડિવાઇસ સાથે શેર કરવા માટે તેને પોર્ટબલ Wi-Fi હૉટસ્પૉટમાં બદલી શકો છો
  • ઉપકરણને પોર્ટેબલ Wi-Fi હોટસ્પોટ તરીકે ઉપયોગ કરતી વખતે Wi-Fi જોડાણ દ્વારા ઇન્ટરનેટ પ્રવેશ કરવા માટે ઉપકરણ એપ્લિકેશનોનો ઉપયોગ કરવામાં અસમર્થ.

  • તમે Wi-Fi હૉટસ્પૉટ સેટિંગ્સમાં પોર્ટબલ Wi-Fi હૉટસ્પૉટ કન્ફિગર કરી શકો છો

  • વધુ માહિતી માટે, http://www.android.com/tether#wifi જુઓ

USB જોડી રહ્યું છે

તમારા કમ્પ્યુટર તમારા ઉપકરણનું ઈન્ટરનેટ જોડાણ શેર કરી શકે છે જો તમે એક USB કેબલ દ્વારા તેમને ટેથર કરો છો તો.
  • ટેથરિંગ વિન્ડોઝ 7, વિન્ડોઝ વિસ્ટા અને લીનક્ષમાં કામ કરે છે

  • ટેથર કરવા પર તમે તમારા કમ્પ્યુટર સાથે SD કાર્ડ જોડી શકતા નથી.

  • અન્ય ઓપરેટિંગ સિસ્ટમ સાથે USB ટેથરિંગનો ઉપયોગ કેવી રીતે કરવો સહિત વધુ માહિતી માટે, http://www.android.com/tether#usb સાઇટમાં જાઓ

બ્લ્યુટુથ ટેથરિંગ

તમે તમારા ઉપકરણના ઈન્ટરનેટ જોડાણને તમારા કોમ્યુટર સાથે શેર કરવા માટે તમારા એન્ડ્રોઈડ ઉપકરણને તમારા કોમ્યુટર સાથે બ્લ્યુટુથ દ્વારા ટેથર કરી શકો છો.
  • બ્લ્યુટુથ ટેથરિંગ તેવા કોઈપણ કોમ્યુટર અથવા ડીવાઈસ સાથે કામ કરે છે જે બ્લ્યુટુથ દ્વારા ઈન્ટરનેટ પ્રવેશ કરી શકે છે

  • વધુ માહિતી માટે http://www.android.com/tether#bluetooth
