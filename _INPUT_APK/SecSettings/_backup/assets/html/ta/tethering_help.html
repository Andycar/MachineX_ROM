கையடக்க Wi-Fi ஹாட்ஸ்பாட்

உங்கள் சாதனத்தை அதன் இணைய இணைப்பை கணினிகள் மற்றும் மற்ற சாதனங்களுடன் பகிர கையடக்க Wi-Fi ஹாட்ஸ்பாட்டாக மாற்றலாம்
  • சாதனம் கையடக்க Wi-Fi ஹாட்ஸ்பாட்டாக செயல்படுகையில் Wi-Fi இணைப்பு வழியே இணையத்தை அணுக சாதன பயன்பாடுகளை பயன்படுத்த இயலாது

  • Wi-Fi ஹாட்ஸ்பாட் அமைவுகளில் கையடக்க Wi-Fi ஹாட்ஸ்பாட்டை கட்டமைக்கலாம்

  • மேலும் விபரத்திற்கு, http://www.android.com/tether#wifi காண்க

USB டித்தரிங்

உங்கள் சாதனங்களை USB கேபிளுடன் ஒன்றாக டித்தர் செய்தால, கணினியுடன் சாதன இணைய இணைப்பை பகிரலாம்
  • வின்டோஸ் 7, வின்டோஸ் விஸ்டா, மற்றும் லினக்ஸ் ஆகியவற்றுடன் டித்தரிங் வேலை செய்கிறது

  • டித்தர் செய்கையில் உங்கள் கணினியுடன் SD அட்டையை இணைக்க முடியாது

  • வேறு இயக்கத்தளங்களுடன் எவ்வாறு USB டித்தரிங்களை பயன்படுத்த வேண்டும் என்பது பற்றி மேலும் அறிய, http://www.android.com/tether#usb செல்க

ப்ளூடூத் டித்தரிங்

உங்கள் கணினியுடன் சாதனத்தின் இணைய இணைப்பைப் பகிர, உங்கள் ஆன்ட்ராய்டு சாதனத்தை உங்கள் கணினியுடன் ப்ளூடூத் வழியே டித்தர் செய்யலாம்
  • ப்ளூடூத் வழியே இணையத்தை அணுகும் அணைத்து சாதனங்களுடன் ப்ளூடூத் டித்தரிங் வேலை செய்யும்

  • மேலும் விபரத்திற்கு, http://www.android.com/tether#bluetooth காண்க
