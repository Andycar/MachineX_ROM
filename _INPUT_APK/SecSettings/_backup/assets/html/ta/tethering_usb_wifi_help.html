USB டித்தரிங்

உங்கள் சாதனங்களை யுஎஸ்பி கேபிளுடன் ஒன்றாக டித்தர் செய்தால, கணினியுடன் சாதனா இணைய இணைப்பை பகிரலாம்
  • வின்டோஸ் 7, வின்டோஸ் விஸ்டா, மற்றும் லினக்ஸ் ஆகியவற்றுடன் டித்தரிங் வேலை செய்கிறது

  • டித்தர் செய்கையில் உங்கள் கணினியுடன் எஸ்டி அட்டையை இணைக்க முடியாது

  • வேறு இயக்கத்தளங்களுடன் எவ்வாறு யுஎஸ்பி டித்தரிங்களை பயன்படுத்த வேண்டும் என்பது பற்றி மேலும் அறிய, http://www.android.com/tether#usb செல்க

கையடக்க Wi-Fi ஹாட்ஸ்பாட்

உங்கள் சாதனத்தை அதன் இணைய இணைப்பை கணினிகள் மற்றும் மற்ற சாதனங்களுடன் பகிர கையடக்க வை-ஃபை ஹாட்ஸ்பாட்டாக மாற்றலாம்
  • சாதனம் கையடக்க வை-ஃபை ஹாட்ஸ்பாட்டாக செயல்படுகையில் வை-ஃபை இணைப்பு வழியே இணையத்தை அணுக சாதன பயன்பாடுகளை பயன்படுத்த இயலாது

  • வை-ஃபை ஹாட்ஸ்பாட் அமைவுகளில் கையடக்க வை-ஃபை ஹாட்ஸ்பாட்டை கட்டமைக்கலாம்

  • மேலும் விபரத்திற்கு, http://www.android.com/tether#wifi காண்க
