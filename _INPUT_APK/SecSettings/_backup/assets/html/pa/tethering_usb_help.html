USB ਟੈਥਰਿੰਗ

ਤੁਹਾਡਾ ਕੰਪਿਊਟਰ ਡਿਵਾਈਸ ਦਾ ਇੰਟਰਨੈੱਟ ਕਨੈਕਸ਼ਨ ਸ਼ੇਅਰ ਕਰ ਸਕਦਾ ਹੈ ਜੇਕਰ ਤੁਸੀਂ ਉਨ੍ਹਾਂ ਨੂੰ USB ਕੇਬਲ ਨਾਲ ਇਕੱਠੇ ਟੈਥਰ ਕਰ ਦਿੰਦੇ ਹੋ
  • ਟੈਥਰਿੰਗ ਵਿੰਡੋਸ 7, ਵਿੰਡੋਸ ਵਿਜ਼ਟਾ, ਅਤੇ ਲੀਨੱਕਸ ਵਿੱਚ ਕੰਮ ਕਰਦੀ ਹੈ

  • ਜਦ ਸੀਮਿਤ ਹੋਵੇ ਤਾਂ ਤੁਸੀਂ SD ਕਾਰਡ ਆਪਣੇ ਕੰਪਿਉਟਰ ਨਾਲ ਕਨੈਕਟ ਨਹੀਂ ਕਰ ਸਕਦੇ

  • ਹੋਰ ਜਾਣਕਾਰੀ ਲਈ, USB ਦੇ ਹੋਰ ਅਪ੍ਰੇਟਿੰਗ ਸਿਸਟਮਸ ਨਾਲ ਕੰਮ ਕਰਨ ਸਮੇਤ, http://www.android.com/tether#usb ਤੇ ਜਾਓ।
