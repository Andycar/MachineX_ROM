ජංගම Wi-Fi කේන්ද්‍රස්ථානය

උපාංගයේ අන්තර්ජාල සම්බන්ධතාවය පරිගණක හෝ වෙනත් උපාංග සමග බෙදාහදා ගැනීමට, ඔබට ඔබේ උපාංගය ජංගම හොට්ස්පොට් එකක් බවට පත් කළ හැක.
  • උපාංගය අතේ ගෙන යා හැකි Wi-Fi කේන්ද්‍රස්ථාන එකක් ලෙස ක්‍රියා කරන අතර Wi-Fi හරහා අන්තර්ජාලය වෙත පිවිසීමට උපාංග යෙදුම් භාවිතා කළ නොහැක.

  • Wi-Fi මොඩමයක් ලෙස අන්තර්ජාලයට සම්බන්ධ කිරීමේ සැකසුම් තුළ ඔබට අතේ ගෙන යා හැකි Wi-Fi මොඩමයක් ලෙස අන්තර්ජාලයට සම්බන්ධ කිරීම වින්‍යාස කළ හැක.

  • වැඩිදුර තොරතුරු සඳහා, http://www.android.com/tether#wifi වෙත පිවිසෙන්න.

USB ටෙදරින්

ඔබට පරිගණකය හා දුරකථනය USB රැහැනක් මගින් සබඳ කල හැකි නම් උපාංගයේ අන්තර්ජාල සබඳතාව බෙදාහදාගත හැක.
  • සම්බන්ධකිරීම Windows 7, Windows Vista හා Linux මත ක්‍රියාකරයි.

  • සම්බන්ධකල විට ඔබට ඔබේ පරිගණකයට SD කාඩ්පත සවිකල නොහැක.

  • USB බැඳීම වෙනත් මෙහයුම් පද්ධති සමඟ ක්‍රියාකරන්නේ කෙසේද වැනි තවත් තොරතුරු සඳහා පිවිසෙන්න http://www.android.com/tether#usb

  • බ්ලූටූත් හරහා අන්තර්ජාලය වෙත ප්‍රවේශ විය හැකි ඕනෑම පරිගණකයක් හෝ උපාංගයක් සමග බ්ලූටූත් ටෙදරින් ක්‍රියා කරයි.

  • වැඩි විස්තර සඳහා පිවිසෙන්න http://www.android.com/tether#bluetooth
