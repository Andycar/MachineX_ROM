USB টেথেৰিং

আপোনাৰ কম্পিউটাৰে আপোনাৰ ডিভাইচৰ ইণ্টাৰনেট সংযোগ ভাগ-বতৰা কৰিব পাৰে যদিহে আপুনি তাক ইউএচবি কেব'লৰ সৈতে সংযোগ কৰে
  • উইন্ডোওজ 7, উইন্ডোওজ ভিস্টা, আৰু লিনাক্সৰ সৈতে সংযোগ কৰা কাৰ্য্য কৰিছে

  • সংযোগ কৰাৰ সময়ত আপুনি আপোনাৰ কম্পিউটাৰলৈ এচডি কাৰ্ড সংযোগ কৰিব নোৱাৰে

  • অন্য অপাৰেটিং ছিষ্টেমৰ সৈতে অন্তৰ্ভুক্ত হৈ থকা ইউএচবি সংযোগকৰণ কেনেকৈ ব্যৱহাৰ কৰিব, তাৰ অধিক তথ্যৰ কাৰণে, http://www.android.com/tether#usb লৈ যাওক

প'ৰ্টেব'ল Wi-Fi হ'টস্প'ট

কম্পিউটাৰ আৰু অন্য ডিভাইচসমূহৰ লগত ইণ্টাৰনেট সংযোগ ভাগ-বতৰা কৰিবৰ বাবে আপুনি আপোনাৰ ডিভাইচক এটা ম'বাইল হ'টস্প'টলৈ ৰূপান্তৰিত কৰিব পাৰে৷
  • ডিভাইচ চাৰ্ভাৰ প'ৰ্টেব'ল Wi-Fi হ'টস্প'টৰ ৰূপত থকাৰ সময়ত Wi-Fi হৈ ইণ্টাৰনেট সংযোগ প্ৰৱিষ্ট কৰি ডিভাইচ এপ্লিকেশ্বন ব্যৱহাৰ কৰিবলৈ অক্ষম৷

  • আপুনি Wi-Fi টিথাৰিং ছেটিংছত পৰ্টেবল Wi-Fi টিথাৰিং কনফিগাৰ কৰিব পাৰে৷

  • অধিক তথ্যৰ কাৰণে, http://www.android.com/tether#wifi চাওক
