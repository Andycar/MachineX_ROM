ການເຊື່ອມໂຍງເຄືອຂ່າຍ USB

ຄອມພິວເຕີຂອງທ່ານສາມາດແຊຣ໌ການເຊື່ອມຕໍ່ອິນເຕີເນັດຂອງອຸປະກອນໄດ້ ຫາກທ່ານເຊື່ອມຕໍ່ໄວ້ກັບ USB
  • ການເຊື່ອມຕໍ່ USB ເຮັດວຽກກັບ Windows 7, Windows Vista ແລະ Linux

  • ທ່ານບໍ່ສາມາດຕໍ່ແຜ່ນ SD ກັບຄອມພິວເຕີຂອງທ່ານໄດ້ເມື່ອເຊື່ອມຕໍ່ຢູ່

  • ສຳລັບຂໍ້ມູນເພີ່ມເຕີມຮວມເຖິງວິທີໃຊ້ການເຊື່ອມຕໍ່ດ້ວຍ USB ກັບລະບົບປະຕິບັດການອື່ນໆໃຫ້ໄປທີ່ http://www.android.com/tether#usb

ຮັອດສະປ໋ອດ Wi-Fi ເຄື່ອນທີ່

ທ່ານສາມາດປ່ຽນອຸປະກອນແອນດຣອຍຂອງທ່ານເປັນຮັອດສະປ໋ອດ Wi-Fi ແບບພົກພາ ເພື່ອການແຊຣ໌ການເຊື່ອມຕໍ່ອິນເຕີເນັດກັບຄອມພິວເຕີຫຼືອຸປະກອນອື່ນ
  • ບໍ່ສາມາດໃຊ້ແອພພລິເຄຊັນອຸປະກອນເພື່ອເຂົ້າສູ່ອິນເຕີເນັດຜ່ານການເຊື່ອມຕໍ່ Wi-Fiໃນຂະນະທີ່ອຸປະກອນເຮັດໜ້າເປັນຮັອດສະປ໋ອດ Wi-Fi ແບບພົກພາ

  • ທ່ານສາມາດກຳນົດຄ່າຮັອດສະປ໋ອດ Wi-F ແບບພົກພາໄດ້ໃນການຕັ້ງຄ່າຮັອດສະປ໋ອດ Wi-Fi

  • ສຳລັບຂໍ້ມູນເພີ່ມເຕີມໃຫ້ໄປທີ່ http://www.android.com/tether#wifi
