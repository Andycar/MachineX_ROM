USB तेथरिंग

युएसबी केबलबरोबर आपण त्यांना एकत्र टेथर केल्यास आपला कॉम्प्यूटर उपकरणाचे इंटरनेट कनेक्शन सहभागी करू शकतो
  • टेथरिंग विंडोस7,विंडोस विस्टा, आणि लिनुक्ष बरोबर काम करते

  • टेथर केलेले असल्यास आपण एसडी कार्ड आपल्या कॉम्प्यूटरला कनेक्ट करू शकत नाही

  • अधिक माहितीसाठी, युएसबी टेथरिंग इतर ऑपरेटिंग सिस्टीमबरोबर कसे वापरावे यासह, अधिक माहितीसाठी, http://www.android.com/tether#usb-वर जा

पोर्टेबल Wi-Fi हॉटस्पॉट

आपण आपले डिव्हाईस, त्याचे इंटरनेट कनेक्शन काँम्पुटर्स किंवा अन्य डिव्हाईसेस बरोबर शेअर करण्यासाठी वाय-फाय हॉटस्पॉटमध्ये बदलू शकता
  • डिव्हाईस वाय-फाय हॉटस्पॉट म्हणून काम करताना वाय-फाय कनेक्शन द्वारे इंटरनेट ऍक्सेस करण्यासाठी डिव्हाईस ऍप्लिकेशन्स वापरण्यास अक्षम

  • आपण वाय-फाय हॉटस्पॉट सेटिंग्जमध्ये पोर्टेबल वाय-फाय हॉटस्पॉट कॉन्फिगर करु शकता

  • अधिक माहितीसाठी http://www.android.com/tether#wifi-ला भेट द्या
