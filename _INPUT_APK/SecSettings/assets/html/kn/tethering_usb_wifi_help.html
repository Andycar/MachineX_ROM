USB ಟೀಥರಿಂಗ್

ಯು‌ಎಸ್‌ಬಿ ಕೇಬಲ್‌ನೊಂದಿಗೆ ನೀವು ಅವುಗಳನ್ನು ಒಟ್ಟಿಗೆ ಟೀಥರ್ ಮಾಡಿದಲ್ಲಿ ನಿಮ್ಮ ಕಂಪ್ಯೂಟರ್ ನಿಮ್ಮ ಸಾಧನದ ಅಂತರ್ಜಾಲ ಸಂಪರ್ಕವನ್ನು ಹಂಚಬಹುದು
  • ಟೀಥರಿಂಗ್ ವಿಂಡೋಸ್ 7, ವಿಂಡೋಸ್ ವಿಸ್ತಾ, ಮತ್ತು ಲಿನುಕ್ಷ ನೊಂದಿಗೆ ಕೆಲಸ ಮಾಡುತ್ತದೆ

  • ಟೀಥರ್ಡ್ ಆಗಿರುವಾಗ ನೀವು ಒಂದು ಎಸ್‌ಡಿ(SD ) ಕಾರ್ಡ್‌ನ್ನು ನಿಮ್ಮ ಕಂಪ್ಯೂಟರ್‌ಗೆ ಸಂಪರ್ಕಿಸಲಾಗುವುದಿಲ್ಲ

  • ಇತರ ಆಪರೇಟಿಂಗ್ ಸಿಸ್ಟಮ್‌ಗಳೊಂದಿಗೆ ಹೇಗೆ ಯು ಟೀಥರಿಂಗ್ ಬಳಸಬೇಕೆನ್ನುವುದನ್ನೊಳಗೊಂಡ, ಹೆಚ್ಚಿನ ಮಾಹಿತಿಗಾಗಿ, http://www.android.com/tether#usb ಗೆ ಹೋಗಿ

ಪೋರ್ಟೆಬಲ್ Wi-Fi ಹಾಟ್‌ಸ್ಪಾಟ್

ಕಂಪ್ಯೂಟರ್‌ಗಳು ಅಥವಾ ಇತರ ಸಾಧನಗಳೊಂದಿಗೆ ಅದರ ಇಂಟರ್‌ನೆಟ್ ಸಂಪರ್ಕ ಹಂಚಲು, ನೀವು ನಿಮ್ಮ ಸಾಧನವನ್ನು ಪೋರ್ಟೆಬಲ್ ವೈ-ಫೈ ಹಾಟ್‌ಸ್ಪಾಟ್ ಆಗಿ ಟರ್ನ್ ಮಾಡಬಹುದು
  • ಸಾಧನವು ಪೋರ್ಟೆಬಲ್ ವೈ-ಫೈ ಹಾಟ್‌ಸ್ಪಾಟ್ ಆಗಿ ಕಾರ್ಯನಿರ್ವಹಿಸುವಾಗ ವೈ-ಫೈ ಸಂಪರ್ಕದ ಮೂಲಕ ಇಂಟರ್‌ನೆಟ್ ಪ್ರವೇಶಿಸಲು ಸಾಧನ ಅನ್ವಹಿಸುವಿಕೆಗಳನ್ನು ಬಳಸಲು ಅಸಮರ್ಥ

  • ವೈ-ಫೈ ಹಾಟ್‌ಸ್ಪಾಟ್ ಸಂಯೋಜನೆಗಳಲ್ಲಿ ನೀವು ಪೋರ್ಟೆಬಲ್ ವೈ-ಫೈ ಹಾಟ್‌ಸ್ಪಾಟ್ ಸಂರಚಿಸಬಹುದು

  • ಹೆಚ್ಚಿನ ಮಾಹಿತಿಗಾಗಿ, http://www.android.com/tether#wifi ಭೇಟಿ ಮಾಡಿ
